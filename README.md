# Tarea1_estructuras_abstractas

## Tarea 1 Lista doblemente enlazada

Modifique el código de Nodo y Lista para obtener una lista doblemente enlazada, donde cada Nodo tenga un puntero tanto al elemento siguiente como al anterior.
Escriba métodos para:
contar: contar el total de elementos en la lista
imprimir: imprimir todos los elementos de la lista, en una dirección (siguiente)
imprimir_reverso: imprimir todos los elementos de la lista, en reverso (anterior)
eliminar: eliminar un elemento de la lista
filtrar: crear una lista nueva, en donde los elementos que contiene esa lista sean mayores a un valor dado (referencia)
Escriba un informe técnico (PDF), en donde explique en una o dos oraciones cómo funciona cada función.
Grabe una demostración en video, en donde muestre las funciones del código que escribió y explique cómo funcionan. El video no debe durar más de 7 minutos.
El código e informe técnico deben subirse a un repositorio de Git nombrado nombres_apellidos_tarea_1, y el video debe subirse a Mediación virtual.
La fecha máxima de entrega de ambas partes será el domingo 19 de abril a las 23:59.
